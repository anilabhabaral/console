Jakarta Bean バリデーションは以下のような Java 仕様です:

  • アノテーションを用いてオブジェクトモデルで制約を表すことができます。
  • 拡張可能な状態でカスタム制約を作成できます。
  • オブジェクトとオブジェクトグラフの検証に API を提供します。
  • メソッドおよびコンストラクターのパラメーターおよび戻り値の検証に API を提供します。
  • 違反を報告します (ローカライズされます)。
  • Java SE で実行され、Jakarta EE 8 に統合されています。

${build.fullName} は、その実装に Hibernate Validator を使用します。
